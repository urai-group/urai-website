# Website of the URAI

## How to contribute

- simply fork this repo, make your changes and open a pull request
- merged changes are automatically deployed by vercel

> Note: Please comply to [commit conventions](https://www.conventionalcommits.org/en/v1.0.0/)!


## Current tech stack 

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
